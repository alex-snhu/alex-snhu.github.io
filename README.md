## Software Engineering Artifact – Travlr Application

### Description

The Travlr application is a full stack web application I developed using the MEAN stack (MongoDB, Express, Angular, Node.js). It includes both a customer-facing site and an admin interface used to manage trip data.

### Enhancements

* Added a search feature so users can find trips by name or destination
* Updated the API to accept search parameters and return filtered results
* Improved how the frontend, backend, and database interact

### Skills Demonstrated

* Full stack development
* API design and backend logic
* MongoDB querying
* Understanding how larger systems connect together

* Improved how users can explore and interact with the data

### Skills Demonstrated

* MongoDB query construction
